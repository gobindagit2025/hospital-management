# Hospital Management System

A web-based Hospital Management System built using Python.

## 📌 Features
- Patient registration
- Doctor management
- Appointment scheduling
- Record management

## 🛠️ Tech Stack
- Python
- Flask / Django (edit based on your project)
- MySQL
- HTML, CSS, JavaScript

## 🚀 How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python app.py   (or manage.py runserver)

## 📂 Project Structure
- models/
- templates/
- static/
- app.py
